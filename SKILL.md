# Vietnamese Language Support Skill

> **OpenClaw Plugin Skill** — Rule-based Vietnamese text correction, language mixing detection, and native fluency enhancement for LLMs.

## 🎯 Mục đích
Giúp bất kỳ model LLM nào (đặc biệt là các model yếu tiếng Việt như Nemotron, Gemma...) sinh ra tiếng Việt **chuẩn chính tả, đúng dấu, tách từ đúng, không lẫn lộn tiếng Trung/Anh vô nghĩa**, và gợi ý diễn đạt tự nhiên như người bản xứ.

## 🚀 Cài đặt & Sử dụng

### Yêu cầu
- Node.js ≥ 18 (ESM support)
- OpenClaw gateway (để hook/plugin discovery)

### Cài đặt trong OpenClaw
```bash
# Copy plugin vào thư mục plugin-skills
cp -r vietnamese-language-support ~/.openclaw/plugin-skills/

# Build
cd ~/.openclaw/plugin-skills/vietnamese-language-support && node build.mjs
```

### Sử dụng qua CLI (`vn-lang`)
```bash
# Fix text
vn-lang fix "Model的 này hơi yếu tiếng Việt的"
# Output: "Model này hơi yếu tiếng Việt" + mixing detected

# Check only (không auto-fix)
vn-lang check "Tôi think rằng này okay"

# Native alternatives
vn-lang native "Tôi nghĩ là"

# Search idioms
vn-lang idiom "kiên trì"

# Regional variant
vn-lang region "now" south
```

### Sử dụng trong code (Module API)
```javascript
import { fixText, checkText, getNativeAlternatives, searchIdioms, getRegionalVariant } from 'vietnamese-language-support';

// Auto-fix
const result = fixText('Model的 này hơi yếu tiếng Việt的');
console.log(result.fixed);        // "Model này hơi yếu tiếng Việt"
console.log(result.corrections);  // spelling/tone/segmentation fixes
console.log(result.mixingIssues); // Chinese chars, English fillers detected

// Check only
const check = checkText('Tôi think rằng này okay');
// { corrections: [], mixingIssues: [{ type: 'english_insertion', token: 'think' }] }

// Native suggestions
getNativeAlternatives('Tôi nghĩ là');
// [{ type: 'formal_to_casual', original: 'Tôi nghĩ là', alternatives: ['Mình nghĩ', 'Em nghĩ', ...] }]

// Idioms
searchIdioms('kiên trì');
// [{ phrase: 'có công mài sắt...', meaning: '...', category: 'persistence' }]

// Regional
getRegionalVariant('now', 'south'); // "bây h/giờ nì"
```

## 🔌 OpenClaw Integration

### Tool Definitions
5 tools có sẵn cho agent gọi:
| Tool | Mô tả | Input | Output |
|------|-------|-------|--------|
| `vn_fix` | Fix full text | `{text, options?}` | `{fixed, corrections, mixingIssues, nativeAlternatives}` |
| `vn_check` | Check issues only | `{text}` | `{corrections, mixingIssues}` |
| `vn_native` | Native alternatives | `{phrase}` | `[{type, original, alternatives, context}]` |
| `vn_idiom` | Search idioms | `{keyword}` | `[{phrase, meaning, english, category, usage}]` |
| `vn_region` | Regional variant | `{word, region}` | `string` |

### Hook: `post_model_output`
Tự động chạy sau khi model sinh output. Nếu phát hiện tiếng Việt, sẽ:
1. Normalize Unicode NFC
2. Fix spelling & tone marks
3. Fix word segmentation
4. Strip Chinese chars & English fillers (giữ whitelist: React, API, TypeScript...)
5. Ghi log structured cho observability

**Cấu hình qua metadata:**
```json
{
  "formalLevel": "auto|formal|casual",
  "vnLangConfig": {
    "whitelist": ["CustomTerm1", "CustomTerm2"],
    "stripMixing": true
  }
}
```

```bash

node -e "import {fixText} from './dist/index.mjs'; console.log(fixText('test text'))"
```

### Add Dictionary Entries
1. Edit file trong `assets/*.json`
2. Rebuild: `node build.mjs`
3. Test: `node src/cli.mjs fix "new entry"`
